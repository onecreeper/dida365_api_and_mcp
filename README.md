# Dida365 API 和 MCP 集成项目

这是一个用于集成滴答清单(Dida365) API 和 Model Context Protocol (MCP) 的Python项目。

## 功能特性

- 提供滴答清单API的Python封装
- 实现MCP服务器接口
- 支持任务同步和管理功能

## 使用方式

cherrystudio

--directory
{路径}
